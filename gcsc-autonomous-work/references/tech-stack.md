# GCSC Pre-Authorized Technology Stack

Serhiy has approved this stack for GCSC and all future projects.

## Frontend
- **HTML5** — semantic markup
- **Tailwind CSS** — utility-first CSS (via CDN)
- **Vanilla JavaScript** — no frameworks needed for MVP
- **Lucide Icons** — clean icon set (via CDN)
- **Google Fonts** — Inter font family
- **GSAP** — scroll animations

## Backend
- **Node.js 20+** — runtime
- **Express.js** — web framework
- **PostgreSQL** — relational database
- **JWT** — authentication tokens
- **bcryptjs** — password hashing
- **Helmet** — security headers
- **express-rate-limit** — DDoS protection
- **cors** — cross-origin requests
- **validator** — input sanitization
- **morgan** — request logging

## Payments
- **Stripe** — credit card processing
- **Stripe Connect** — contractor payouts
- **Stripe Webhooks** — payment confirmations

## Blockchain
- **XPR Network** — escrow smart contracts
- **@proton/web-sdk** — wallet connection
- **@proton/api** — blockchain queries
- **WebAuth** — user wallet authentication

## Security
- **AES-256-GCM** — data encryption
- **PBKDF2** — key derivation (100k iterations)
- **RIPEMD160** — address hashing (npm package)
- **secp256k1** — key pair generation

## Communication
- **Gmail API** — OTP email delivery
- **Twilio** — SMS OTP backup
- **Google Drive** — encrypted key vault

## Hosting
- **GitHub Pages** — static frontend
- **Render.com** — Node.js backend
- **Custom domains** — gcsc.store, xprnet.org

## DevOps
- **Docker** — containerization
- **Git** — version control
- **GitHub** — repository + CI/CD

## Tools (Pre-Approved)
- MCPSafe.io — security scanning
- Semgrep — code security analysis
- SafeDep Vet — dependency scanning
- Lighthouse — performance testing
